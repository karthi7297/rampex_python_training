# Python Training Program

A comprehensive Python learning program covering fundamental concepts through advanced data structures.

## 📚 Program Structure

### Basics & Fundamentals

Core Python concepts and syntax:

- **ascii.py** - Working with ASCII values and character encoding
- **casting.py** - Type casting and data type conversions
- **condition.py** - Conditional statements (if/elif/else)
- **condition2.py** - Advanced conditional logic
- **swap.py** - Swapping variables and values

### Control Flow & Problem Solving

Conditional logic and decision-making:

- **largest.py** - Finding the largest number
- **leafyear.py** - Leap year calculation logic
- **star.py** - Pattern printing with stars
- **numcheck.py** - Number validation checks

### Loops & Mathematical Operations

Working with loops and calculations:

- **interst_cal.py** - Interest calculations
- **reverce_num.py** - Reversing numbers
- **revercewith_loop.py** - Reversing numbers using loops
- **fact.py** - Factorial calculations
- **multiTable.py** - Multiplication table generation
- **root.py** - Root calculations (square root, etc.)
- **sumAvg.py** - Sum and average calculations

### String Operations

Working with strings and text:

- **Vowel.py** - Vowel counting and detection
- **countvowels.py** - Count vowels in strings
- **palindrome.py** - Palindrome checking
- **str_rev.py** - String reversal
- **Ecount_Ocount.py** - Character counting

### Number Operations

Numeric calculations and manipulations:

- **even_num.py** - Even number operations
- **Natural_Num.py** - Natural number operations
- **numrev.py** - Number reversal practice
- **secondlargest.py** - Finding second largest element

### Functions

Reusable code and function concepts:

- **EvenOdd_fun.py** - Function to check even/odd numbers
- **prime.py** - Prime number checking function

### Data Structures

Working with Python's collection data types:

**Lists:**

- **list_len.py** - List length and operations
- **listRev.py** - Reversing lists
- **margeList.py** - Merging lists
- **oddEvenCount.py** - Counting odd and even in lists
- **remove_dup.py** - Removing duplicates from lists
- **SaparateOddEven.py** - Separating odd and even numbers
- **sum.py** - Sum of list elements
- **min_num.py** - Finding minimum number
- **largest.py** - Finding largest element

**Tuples:**

- **convertTuble.py** - Converting to tuples
- **IterateTuble.py** - Iterating through tuples
- **tubleRev.py** - Reversing tuples
- **unpack.py** - Tuple unpacking

**Dictionaries:**

- **create_dic.py** - Creating and initializing dictionaries
- **count_dic.py** - Counting elements in dictionaries
- **updata_dic.py** - Updating dictionary values
- **sqrt_dic.py** - Dictionary operations with square roots
- **use_itemKey.py** - Using dictionary items and keys
- **elementExit.py** - Checking element existence

**Sets:**

- **elementExists_set.py** - Checking set membership
- **elementCount.py** - Counting set elements
- **differenceOf_set.py** - Set difference operations
- **intersectionOf_set.py** - Set intersection operations
- **unionOf_set.py** - Set union operations
- **symmetricDiff_set.py** - Symmetric difference of sets

## 🎯 Learning Path

1. **Basics**: Master Python syntax, data types, and control flow
2. **Loops & Math**: Practice loops and mathematical operations
3. **Functions**: Learn functions and code reusability
4. **Data Structures**: Understand Python's built-in data structures (lists, tuples, dictionaries, sets)
